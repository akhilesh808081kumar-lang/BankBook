# BankBook - Java Bank Simulator
  
  A console-based banking system built in Java demonstrating OOP concepts.
  
  ## Features
  - Create account, Deposit, Withdraw, Check Balance, Transfer
  
  ## Tech Stack
  - Java, Scanner, ArrayList, OOP (Classes, Objects)
  
  ## How to Run
  1. Clone repo: `git clone <link>`
  2. Compile: `javac BankBook.java`
  3. Run: `java BankBook`
